ES6 对象新特性

  1. 属性简写：如果属性名和表示属性值的变量名相同，可以省略属性值，
let obj = {name:'name', age:'age'} => let obj = {name, age}
  2. 计算属性名：可以使用表达式作为属性名(需放在方括号内)
let obj = {'HuaHua_Age':18} => let obj = {[name + '_Age']:18}
  3. 方法简写：可以省略方法的冒号和function关键字
let obj = {
  sayHello : function(){
    console.log('hello')
  }
}  =>
let obj = {
  sayHello() {
    console.log('hello')
  }
}
  4. 对象扩展运算符：可以使用`...`运算符将一个对象的所有可枚举属性复制到另一个对象中
let obj1 = {a: 1, b: 2}; let obj2 = {...obj1, c: 3}`
  5. Object.assign()方法：可以用来合并多个对象的可遍历属性，例如
let obj1 = {a: 1,b: 2};
let obj2 = {c: 3, d: 4};
let obj3 = Object.assign(obj1, obj2)
  6. Object.is()方法：可以用来判断两个值是否严格相等(与（===）基本类似。)
Object.is("q","q");      // true
Object.is(1,1);          // true
Object.is([1],[1]);      // false
Object.is({q:1},{q:1});  // false

与（===）的区别

//+0不等于-0
Object.is(+0,-0);  //false
+0 === -0  //true
//NaN等于本身
Object.is(NaN,NaN); //true
NaN === NaN  //false

  7. Object.keys()、Object.values()、Object.entries()方法：可以用来获取对象的键、值、键值对数组
let obj = {a: 1, b: 2}
Object.keys(obj) // ['a', 'b']
